# Nilesh Patwardhan Portfolio with Certificates

Updated professional portfolio for Nilesh Ashok Patwardhan.

## Included
- One-page portfolio website
- Certificates and qualification PDF links
- Hanwha Vision certificate image
- Professional profile image placeholder

## Important profile photo note
The real profile photo was not available as a separate sandbox file when this package was generated.
To add it manually:
1. Save the real profile photo as `profile.jpg`
2. Upload it to the `assets` folder
3. In `index.html`, replace:
   assets/profile-placeholder.svg
   with:
   assets/profile.jpg
